# Agentic Honey-Pot for Scam Detection

## Team Members
SHIVANI KHICHI
ADITYA GUPTA
YOKESH

## Problem Statement
This project builds an AI-powered agentic honeypot system that detects scam messages and autonomously engages scammers to extract actionable intelligence such as bank account numbers, UPI IDs, and phishing links.

## Objective
- Detect scam intent from incoming messages
- Engage scammers using a human-like AI agent
- Extract scam intelligence
- Return structured JSON output via a public API
